# Naila Yaqoob — AI Developer Portfolio

A high-converting, single-page freelance portfolio website built with pure HTML, CSS, and JavaScript.

## Stack
- HTML5 / CSS3 / Vanilla JS (no frameworks)
- Google Fonts: Inter + JetBrains Mono
- Glassmorphism dark theme with neon purple/cyan accents

## Run
Open `index.html` directly in any browser. No build step needed.

## Sections
1. Hero with typing animation + particle background
2. About with animated stats
3. Services (6 AI service cards)
4. Projects / Case Studies (featured + 3 cards)
5. Why Choose Me
6. Fiverr Gigs + LinkedIn Tagline
7. Contact form
8. Footer
